# Plan 18-5: Visual Hierarchy & Modern Aesthetic

## Objective
Modernize the visual design with glassmorphism-inspired panels, custom-styled controls, accent colors for primary actions, and consistent spacing. Create clear visual hierarchy between Quick Mode (primary), room animation (secondary), and settings (configuration).

## Tasks

### T15 — Visual Differentiation
- Quick Mode panel: stronger glow border, slightly lighter background, making it visually "primary"
- Settings defaults sections: add a subtle "Defaults" badge/label near the heading
- Dashboard override section ("Advanced overrides"): add a subtle "Override" badge/label
- Live Editor: distinguish from the running list with a slightly different background

### T16 — Modern Button & Control Styling
- Buttons: refined gradients (less harsh), subtle backdrop-blur, smoother hover transitions
- Active/selected state: glow ring effect (box-shadow with accent color)
- Danger buttons: clearer contrast, icon-like visual weight
- Start button: larger (full-width or nearly), accent green color, stands out
- Disabled state: clearly grayed out with reduced opacity
- Focus-visible: visible outline for keyboard navigation

### T17 — Modern Panel & Input Styling
- Panels: softer borders (rgba with lower opacity), subtle inner glow, slight backdrop-blur
- Select/Input: modernized focus ring (accent color glow), subtle transition on focus
- Range sliders: custom CSS styling with accent-colored thumb and track
- Consistent spacing: 8px base grid (gaps, padding, margins use multiples of 0.5rem)
- Heading hierarchy: h2 in panels consistent size, color, weight

## Must-haves
- Quick Mode panel is visually the most prominent section
- Custom-styled range sliders (not browser default)
- Buttons have smooth hover/focus transitions
- Start button is visually prominent with accent color
- Panels have modern glassmorphism-inspired styling
- No functionality changes — pure CSS/visual
