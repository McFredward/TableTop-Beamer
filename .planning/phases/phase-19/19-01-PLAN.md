# Plan 19-1: Align Mode Visibility + Play Area Display + Output Path

## Objective
Make Align Mode easily accessible from the Dashboard, show Play Area boundaries during alignment on the output, and change the output route from /output/final to /output.

## Tasks

### T1 — Align Mode Button Promotion
- Move the Align Mode button from inside the "Active animations" panel to a more visible location
- Place it in the Quick Mode panel area or directly below it as a standalone button
- Style it distinctly: when OFF, subtle but visible; when ON, prominent with glow
- Button text: "Align: OFF" / "Align: ON" (shorter than current "Align mode: OFF")
- The button must be visible on the Dashboard without scrolling on both desktop and mobile
- Remove the old button from the Active Animations panel (avoid duplication)

### T2 — Play Area Display in Align Mode
- When Align Mode is active on /output, render the Play Area polygon boundaries
- Use the existing room-overlay SVG: on /output during align mode, the room-overlay is already shown (CSS rule `body[data-output-role="final-output"].align-mode-active #room-overlay`)
- Add Play Area polygon rendering to the overlay during align mode:
  - Draw each Play Area as a polygon with a distinct stroke color (blue/purple, matching the Settings accent)
  - Use dashed stroke to differentiate from room polygons
  - Only render during align mode — remove when align mode turns off
- The renderRoomOverlay function already runs during align mode on /output — extend it to include Play Areas when align mode is active

### T3 — Output Path Change: /output/final → /output
- In server.mjs: add route handler for `/output` that serves index.html (same as current `/output/final`)
- Keep `/output/final` as a backward-compatible alias (redirect or serve same content)
- In runtime-env.js: update `resolveOutputRole()` to recognize `/output` path as OUTPUT_ROLE_FINAL
- Update any hardcoded `/output/final` references in JS to also check `/output`
- Update the Dashboard footer hint text to reference `/output` instead of `/output/final`

## Must-haves
- Align Mode button visible on Dashboard without scrolling
- Play Area boundaries visible on /output during align mode
- /output serves the output view
- /output/final still works (backward compat)
- No functional regressions in align mode toggle or room overlay
